# A股1日交易策略研究报告

## 概述

本报告研究了一个基于XGBoost的A股1日交易策略。策略在T日收盘后生成信号，T+1日开盘买入，T+2日收盘卖出，严格遵守T+1交易规则和涨跌停限制。

**核心发现**: 通过加入跳空过滤条件（gap 2%-6%），策略从CAGR 27%提升至34.5%，Sharpe从1.21提升至3.08，MaxDD从-40.5%降至-8.8%。

## 策略核心逻辑

### 目标定义
- **预测目标**: `return_1d_open = (T+2 close - T+1 open) / T+1 open`
- **分类标签**: `return_1d_open > 1%` 为正类
- **关键**: 目标从T+1开盘价计算（而非T日收盘价），避免捕获不可交易的隔夜跳空收益

### 交易规则
- **信号时间**: T日收盘后
- **买入时间**: T+1日开盘（以开盘价成交）
- **卖出时间**: T+2日收盘（以收盘价成交）
- **持仓天数**: 2个交易日
- **T+1约束**: 买入当天（T+1）不可卖出
- **涨跌停过滤**: 
  - T日涨停的股票不买入（信号日已涨停，次日大概率继续涨停无法买入）
  - T+1日涨停的股票不买入（开盘涨停无法成交）
  - 跌停日不强制卖出（无法成交），延迟到可卖出时再卖

### ⭐ 跳空过滤（关键改进）
- **定义**: gap = (T+1 open - T close) / T close，即T+1开盘价相对T日收盘价的跳空幅度
- **最优条件**: 只买入 gap 在 2%-6% 之间的股票
- **原理**: 
  - gap < 0%（低开）: 后续大概率继续下跌，胜率仅42-54%
  - gap 0-2%（平开）: 胜率约49%，接近随机
  - gap 2-5%（温和高开）: 动量延续，胜率66.8%
  - gap 5-6%（强势高开）: 动量最强，胜率75.8%
  - gap 6%以上: 几乎全部涨停无法买入，或买入后亏损
  - gap ≥ 9.5%: 100%涨停，无法执行

### 仓位管理
- 每日选概率最高的 `max_pos` 只股票
- 每只股票仓位 = `1 / (hold_days × max_pos)`
- 例: max_pos=3, hold_days=2 → 每只仓位 1/6 ≈ 16.7%
- 最大同时持仓 = hold_days × max_pos = 6只

### 交易成本
- 买入成本: 0.1%（佣金）
- 卖出成本: 0.1%（佣金 + 印花税）
- 往返成本: 0.2%

## 模型

### 特征 (49个)
所有特征均为T日收盘时可获得的信息，无未来函数：

| 类别 | 特征 | 说明 |
|------|------|------|
| 价格 | pct_chg, change, amplitude, body_size, upper_shadow, lower_shadow, is_yang, gap, close_to_high, close_to_low | K线形态 |
| 成交量 | vol, amount, vol_ratio, vol_amount | 量能 |
| 动量 | mom_5d, mom_10d, mom_20d, mom_60d | N日动量 |
| 波动率 | vol_5d, vol_10d, vol_20d | N日波动率 |
| 基本面 | pe, pb, circ_mv, turnover_rate, volume_ratio, log_circ_mv, log_pe, log_pb | 估值/流动性 |
| 资金流 | net_mf_amount, net_mf_vol, buy_lg_amount, sell_lg_amount, buy_elg_amount, sell_elg_amount, net_mf_amount_norm | 主力资金 |
| 技术指标 | rsi_14, kdj_k, bb_position, atr_14, macd | 超买超卖 |
| 新闻 | new_gs, new_bs, new_gi, ths_hot, ths_hot_rank, news_stock_impact, news_market_impact, news_has_mention | 舆情热度 |

### 模型参数
- XGBClassifier: n_estimators=100, max_depth=5, learning_rate=0.1
- subsample=0.8, colsample_bytree=0.8
- tree_method='hist'

### Walk-Forward验证
- **月度滚动**: 每月重新训练模型
- **训练集**: 2020年1月至预测月前一个月的所有数据
- **预测集**: 当月所有交易日
- **预测期间**: 2022年1月 - 2026年3月（51个月）
- **无数据泄露**: 训练数据严格在预测数据之前

## 审计与风险

### 已验证无问题的项目 ✅
1. **Target定义**: `return_1d_open = (T+2 close - T+1 open) / T+1 open`，手动验证正确
2. **特征无未来函数**: 所有49个特征均为T日收盘时已知信息
3. **Walk-Forward无时间泄露**: 训练数据严格在预测月之前
4. **T+1约束**: 买入日不可卖出，已正确实现
5. **涨跌停过滤**: T日/T+1日涨停不买入，跌停日不强制卖出
6. **退市股**: 已退市股票仍保留在历史数据中，无幸存者偏差

### 已知风险与局限 ⚠️

1. **Test期间(2025-2026)结果异常偏高**
   - gap 2-6%策略Test CAGR 50.3% vs Val 28.8%
   - 可能原因: 2025-2026年A股小盘股行情好、样本量小（15个月）
   - **不可外推**: 未来不太可能持续

2. **gap过滤条件本身需要T+1开盘价**
   - gap = (T+1 open - T close) / T close，需要等到T+1开盘才能计算
   - 实际操作: T日收盘后选股，T+1开盘观察gap，符合条件的才下单
   - 这意味着需要实时监控开盘价，增加了操作复杂度
   - **替代方案**: 可以用集合竞价数据预估gap

3. **gap 2-6%过滤后交易频率降低**
   - 460笔执行交易 / 1011个交易日 ≈ 0.45笔/天
   - 部分交易日可能无符合条件的交易
   - 资金利用率可能不足

4. **交易成本可能偏低**
   - 当前0.2%往返成本对大盘股合理
   - 选中股票平均换手率454%（远高于市场平均254%），滑点可能更大
   - 建议对小盘高换手股票使用0.3-0.5%往返成本

5. **模型本质是动量策略**
   - 选中股票T日平均涨幅3.84%（全市场0.07%）
   - 策略收益高度依赖动量因子持续有效
   - 动量反转时策略可能大幅亏损

6. **2026年数据仅3个月**
   - 样本量太小，统计意义有限
   - 需要更长的样本外验证期

### 真实收益预期
- **乐观**: CAGR 25-35%（gap过滤后回测结果）
- **中性**: CAGR 15-25%（考虑滑点、冲击成本、动量衰减）
- **悲观**: CAGR 5-15%（动量因子失效期）

## 待改进方向

1. **集合竞价预估gap**: 用9:15-9:25集合竞价数据预估T+1开盘价，提前判断是否符合gap条件
2. **滑点模型**: 根据换手率/市值动态调整交易成本
3. **仓位管理**: 根据波动率/市场状态调整仓位
4. **多因子融合**: 结合价值/质量因子，减少纯动量依赖
5. **更长的样本外验证**: 至少需要2年以上的独立测试期
6. **日内执行优化**: 从开盘买入改为尾盘买入，减少日内波动风险
