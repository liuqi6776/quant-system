一、核心理念

不预测、不选股、不追 alpha。 用量化做的是三件事:

  1. 分散 —— 把钱分到相关性低的资产(A股 / 黄金 / 海外 / 债券),没有任何单一资产能主导组合。
  2. 风险平价 —— 按"波动率倒数"配权,让每个资产贡献相近的风险,而不是相近的金额。
  3. 控回撤 —— 波动目标 + 趋势减仓 + 高估值买保险,确保熊市拿得住、不割肉。
收益从哪来? 来自"中国及全球资产的长期 beta + 纪律执行(低买高卖、跌破减仓)",而不是"战胜市场"。这是真实数据反复验证、可持续、不会被淘汰的收益来源。

四、具体交易规则(精确到数字)

下面把每一步的触发条件量化到"看到数字就能下单"。所有数字均来自真实回测参数。

① 基础权重:风险平价(逆波动)

每周五收盘,算每个资产过去 60 个交易日的日收益波动率,权重 = (1/波动率) ÷ Σ(1/波动率)。波动越大的资产配得越少。典型基准权重(随市场动态变化,仅示意):

资产典型波动率(年化)→ 典型基础权重
国债ETF 511010~2%~45%
黄金ETF 518880~14%~13%
纳指ETF 513100~20%~9%
沪深300ETF 510300~20%~9%
中证500ETF 510500~22%~8%

注:剩余由波动目标层决定多少进现金(见④)。波动率每周重算,权重随之微调。

② 估值择时:便宜加仓 / 贵减仓(仅对 A股 510300、510500)

用 ERP(股债性价比)= 指数盈利收益率(100/PE) − 10年国债收益率,算它在过去5年(1250交易日)的滚动分位。分位越低 = 越便宜。然后调权重:

权重 × ( 1 − 0.4 × (估值分位 − 0.5) )

估值分位含义权重调整操作
0%(极度低估)5年最便宜× 1.20(加20%)重仓
25%偏低× 1.10加仓
50%(中性)居中× 1.00持平
75%偏贵× 0.90减仓
100%(极度高估)5年最贵× 0.80(减20%)轻仓

③ 趋势闸:跌破年线减半

  • 若 收盘价 ≥ 200日均线(年线) → 权重不变(× 1.0)
  • 若 收盘价 < 200日均线 → 权重 × 0.5(减半,避免接飞刀)
  • 极便宜豁免:A股权益若估值分位 ≤ 20%(历史大底),即使跌破年线也不减半(此时该抄底,不该踏空)

④ 波动目标:组合年化波动上限 10%

用①②③定好的权重,算整个组合的预期年化波动。然后:

缩放系数 = min( 1.0 , 10% ÷ 组合预期波动 )

  • 组合波动 < 10% → 缩放=1.0,满仓配置(本策略大多数时间如此,实际波动仅~4-5%)
  • 组合波动 > 10% → 按比例缩仓,剩余部分放现金(吃 ~2% 货币利息)
  • 缩放系数封顶 1.0 = 永不加杠杆(真实数据证明加杠杆放大风险不增收益)

⑤ Put 保险:高估值时买认沽(需期权权限)

  • 触发:每 20 个交易日检查一次,A股权益(510300/510500)估值分位 > 70% 且当前持有该资产时买入
  • 买什么:对应 ETF 期权,行权价 = 现价 × 0.95(5%虚值认沽),近月(~20交易日到期)
  • 数量:覆盖该 ETF 持仓市值(1张=10000份,按持仓份数折算张数)
  • 到期:结算盈亏后,若仍满足触发条件则滚动续保;否则不再买
  • 估值分位 ≤ 70% → 不买保险(便宜时跌不深,省保险费)

⑥ 调仓阈值:偏离 10% 才动

  • 每周五收盘算出目标权重(经①~④),周一开盘执行
  • 只有当某资产"实际仓位 − 目标仓位"绝对值 > 10% 才调仓,否则全部不动(省成本、防手痒)
  • 交易成本按单边 0.05% 估算(ETF 无印花税,比股票低)
一句话流程: 周五收盘 → 算60日波动定基础权重 → 按5年估值分位加减A股 → 跌破年线的减半(极便宜除外)→ 组合波动压到10%以内、多余进现金 → 估值分位>70%的A股买5%虚值近月认沽 → 偏离>10%的资产周一开盘调仓。
